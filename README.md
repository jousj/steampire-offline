# Стимпериум ОФФЛАЙН

Локальная версия браузерной игры **Стимпериум** (Steampire) от Red Spell.  
Проект позволяет запустить игру полностью офлайн — без интернета и официальных серверов.

---

## 📦 Состав

- Сервер на Python (локальный бэкенд)
- Клиент на Flash (оригинальный .swf)
- X-Chrome — браузер с поддержкой Flash
- Все ресурсы игры (картинки, звуки, локализация)
- Конфигурационные файлы и дампы

---

## 🚀 Запуск

1. Установи **Adobe Flash Player** (если не установлен).  
   Скачать: https://www.softportal.com/software-842-adobe-flash-player.html

2. Распакуй архив в любую папку.

3. Запусти сервер:  
   - **Windows**: дважды кликни по `server.py`  
   - **или** через командную строку:  
python server.py

4. Открой **X-Chrome** (находится в папке) и перетащи в него файл `index.html`.

Игра запустится локально. Сервер должен быть включён во время игры.

---

## ⚠️ Важно

- Обычные браузеры (Chrome, Edge, Firefox) **не поддерживают Flash** — используй только тот, что приложен.
- Игра работает только при запущенном сервере.
- Данные сохраняются локально в JSON-файлах.

---

## 🛠️ Для разработчиков

- Протокол восстановлен по дампам через Charles Proxy.
- Структура пакетов описана в файлах сервера.
- Логи сервера пишутся в `game.log`.
