# Стимпериум ОФФЛАЙН

Локальная версия браузерной игры **Стимпериум** (Steampire) от Red Spell.  
Проект позволяет запустить игру полностью офлайн — без интернета и официальных серверов.

---

## 📦 Состав

- Сервер на Python (локальный бэкенд)
- Клиент на Flash (оригинальный .swf)
- Браузер с поддержкой Flash
- Все ресурсы игры (картинки, звуки, локализация)
- Конфигурационные файлы и дампы

---

## 🚀 Запуск

1. Установите проект

2. Распакуйте в любое удобное вам место

3. Откройте server.py через командную строку (либо двойным кликом по файлу)

4. Распакуйте архив `x64.zip` в котором Браузер

5. Запустите X-Chrome.exe

6. Введите в X-Chrome.exe адрес: `localhost/index.html`

7. Наслаждайтесь игрой! По любым вопросам в Issues либо в контактные данные.

---

## ⚠️ Важно

- Обычные браузеры (Chrome, Edge, Firefox) **не поддерживают Flash** — используйте только тот, что приложен.
- Игра работает только при запущенном сервере.
- Данные сохраняются локально в JSON-файлах.

---

## 🛠️ Для разработчиков

- Код имеет комментарии
- Локализация в nullLangRU.json
- Логика взаимодействий handlers/p10_1.py
- base_state.json - конфигурация базы
- player_state.json - профиль игрока
- остальные файлы с config - настраиваемые
